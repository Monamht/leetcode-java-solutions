# 📘 What is an Array?

An Array is a linear data structure used to store multiple elements of the same datatype in contiguous memory locations.

Each element is accessed using an index.

---

# ✨ Features of Arrays

✅ Fast index-based access  
✅ Contiguous memory allocation  
✅ Easy traversal  
✅ Efficient for storing fixed-size data  

---

# ⚔️ Arrays vs Linked Lists

| Arrays 📚 | Linked Lists 🔗 |
|---|---|
| Contiguous memory allocation | Non-contiguous memory allocation |
| Fixed size | Dynamic size |
| Faster random access | Sequential access |
| Less memory usage | Extra memory required for references |

---

# ☕ Java Array Example

```java
public class Main {

    public static void main(String[] args) {

        // Creating an array
        int[] arr = {2, 4, 6, 8};

        // Accessing elements using index
        System.out.println(arr[0]);

        // Traversing array
        for(int i = 0; i < arr.length; i++) {
            System.out.println(arr[i]);
        }
    }
}
```
